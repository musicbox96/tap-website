desc: "Project Impulse is a compact spring-powered jumping robot designed to simulate a human foot-like launch mechanism. The robot uses a motor-driven winch system to compress high-force springs and release stored energy for controlled vertical jumps. A 3D-printed frame and linear rail system ensure stable and repeatable motion. The system is powered by a LiPo battery and controlled using a Raspberry Pi Pico, with a BTS7960 motor driver handling high-current motor control. A buck converter regulates voltage for safe electronics operation. Future enhancements include AI voice-based control using a Raspberry Pi Zero 2W, directional movement using servo-actuated joints, and stabilization/turning using ducted fans."
