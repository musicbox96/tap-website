## Project Overview

MoodyPi is an interactive outreach project designed to make artificial intelligence and embedded systems more approachable for students and community members. Using a Raspberry Pi, camera, and machine learning model, the system detects a user’s facial emotion and prints a positive message in real time.

## Outreach Activities

**TAP Expo, Georgia Gwinnett College**  
  Promoted the TAP program and demonstrated how AI can be used in interactive projects.

**Atlanta Science Festival, Piedmont Park**  
  Showcased MoodyPi to children and families to encourage interest in STEM and technology.

**Super Saturday STEM Outreach Series**  
  Engaged middle school students in hands-on learning and introduced them to IT and STEM careers.

**STARS Research Showcase**  
  Presented the project in an academic setting to highlight both technical work and impact.

**CREATE Symposium**  
  Shared the project as part of undergraduate research and creative activities.

**Classroom Workshops**  
  Led sessions for non-IT students to explore AI and computer vision concepts.

## How It Works

**1.** The camera captures a live image of the user  
**2.** The system processes the image using a trained model  
**3.** The model predicts the user’s emotion  
**4.** A message is selected based on the emotion  
**5.** The printer prints the message  

## Steps to Use MoodyPi

### 1. Set Up the Hardware

- Connect the Raspberry Pi to power  
- Attach the camera module  
- Connect the thermal printer  

### 2. **Run the Program**

   - Open the terminal  
   - Navigate to the project folder  
   - Run the command below  

```bash
python main.py
```

### 3. Interact
- Stand in front of the camera  
- Wait for detection  
- Receive a printed message
